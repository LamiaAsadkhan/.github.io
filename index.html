Project Overview

This interactive dashboard visualizes the impact of climate change on agricultural productivity across different regions and time periods. Users can explore trends, compare variables, and analyze how climate factors influence crop yields.

Dataset Used

The dataset includes key climate variables such as temperature and rainfall along with agricultural productivity indicators. It supports analysis of climate-driven changes in agricultural output.

Tools & Technologies

  • Tableau Public – dashboard creation and visualization
  • GitHub Pages – project hosting
